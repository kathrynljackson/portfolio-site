To Do List

HTML, CSS, JavaScript

This application allows tasks to be entered into a to-do list and stored for later use. The user can both check off and delete tasks. By filtering the tasks, the user can view completed, incomplete, and all tasks. Due to local storage implementation, the tasks remain on the local computer after refreshing the browser. This application was created for independent practice in about an hour and a half.
